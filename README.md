# Calcula-Tabuada-Melhorado

Projeto proposto no curso oracle one da alura, onde é passado um código em javascript e é proposto ao aluno simplificar a legibilidade do código através de funções
para que qualquer um que leia consiga entender o que foi feito no código, independente de estar comentado ou não.

Ao final do código do arquivo script está comentado o código passado no curso para que fosse melhorado.
